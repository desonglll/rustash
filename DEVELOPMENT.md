# rustash 开发计划

## 核心目标
用 Tauri 重写 Stash 媒体管理系统，采用小步迭代开发。

## 技术栈
- **前端**: React 18 + TailwindCSS 3 + TypeScript
- **后端**: Tauri 2 + Rust
- **数据库**: SQLite + sqlx

## 功能范围（MVP）
1. 影片库管理（CRUD）
2. 文件扫描（本地影片文件）
3. 基础元数据（标题、时长、分辨率）
4. 标签系统
5. 播放功能
